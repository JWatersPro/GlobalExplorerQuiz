Global Explorer Quiz
Interactive geography quiz game built for Harmony Day celebrations. Players guess countries from randomized hints within 3 attempts.

Features
- Random country selection from predefined list
- Hint system (reads facts from country-specific .txt files)
- 3-attempt limit per game
- Persistent scoreboard (results.txt)
- Player name/initial validation
- Ready check with delay option

How to Play
- Enter first name and last initial
- Answer "yes" when ready (or "no" for 3-second delay)
- Guess the country from 1 random hint (3 attempts max)
- View scoreboard after game
